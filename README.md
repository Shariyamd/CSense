# 🚀 CSense – AI-Powered Customer Success Action Platform

CSense is a multi-agent AI platform that helps Customer Success Managers proactively reduce churn, identify growth opportunities, and automate customer engagement workflows.

## ✨ Features

- 🤖 Multi-agent AI workflow using LangGraph
- 📊 Predicts churn risk and expansion opportunities
- 💡 Generates explainable Next Best Actions with AI Confidence Review
- 📈 Interactive dashboard for customer health insights
- 📧 Auto-generates personalized customer emails
- 🎥 Schedules Zoom meetings automatically
- 📝 Creates AI meeting summaries and action items
- 🔄 Updates customer records after meetings

## 🛠 Tech Stack

**Frontend:** React.js, Tailwind CSS  
**Backend:** FastAPI, Python  
**AI:** Gemini API, LangGraph  
**Automation:** Zoom API, Recall.ai, REST APIs

## 🚀 Getting Started

### Backend

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Open: **http://localhost:5173**
